##  Установка

Склонируйте репозиторий и установите зависимости:

```bash
git clone https://github.com/perfilovv/test-map.git
cd react-map-app
npm install
```

##  Запуск

Локальный запуск в режиме разработки:

```bash
npm run dev
```

Локальный запуск собранного проекта:

```bash
git switch gh-pages
cd react-map-app
npm install
npm run preview
```
